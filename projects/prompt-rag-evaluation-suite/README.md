# Prompt and RAG Evaluation Suite

## Goal

Compare prompt designs, retrieval settings, and answer-quality tradeoffs using
repeatable evaluation datasets and measurable quality metrics.

## Architecture Pattern

- Golden question set
- Expected source document mapping
- Retrieval experiment configuration
- Prompt variant registry
- Automated metric calculation
- Human review for answer quality and hallucination risk

## Metrics

- Recall@k
- Precision@k
- Reciprocal rank
- Citation support rate
- Latency
- Human quality rating
