# Explainable Toxicity and Behavior Highlighting

## Goal

Make text classification outputs auditable by highlighting evidence spans and
sentence-level signals behind behavior or toxicity predictions.

## Architecture Pattern

- Text segmentation
- Classifier scoring
- Evidence span extraction
- Sentence-level toxicity scoring
- Reviewer-facing explanation payload

## Review Value

- Faster reviewer triage
- Better trust in model decisions
- Clear audit record for compliance workflows
