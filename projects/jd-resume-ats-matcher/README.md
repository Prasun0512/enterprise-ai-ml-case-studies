# JD-Resume ATS Matcher

## Goal

Compare resumes with job descriptions and produce explainable match scores,
matched skills, missing skills, and role-fit summaries.

## Architecture Pattern

- Resume parsing
- JD parsing
- Skill and title normalization
- Keyword and embedding similarity scoring
- Weighted scoring for must-have and preferred criteria
- Explainable output for recruiters and candidates

## Output Examples

- Overall match score
- Matched skills
- Missing skills
- Experience alignment
- Suggested resume improvements
