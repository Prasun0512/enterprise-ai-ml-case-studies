# AI Learning Content Generator

## Goal

Generate instructional content, quizzes, assessments, and learning summaries
aligned to audience level and learning objectives.

## Architecture Pattern

- Learning objective intake
- Domain knowledge retrieval
- Prompt templates by content type
- Content generation with guardrails
- Rubric-based quality checks
- Human review for pedagogy and factual accuracy

## Quality Controls

- Source-grounded generation
- Bloom-level alignment checks
- Duplicate question detection
- Hallucination review
- Human approval before publishing
